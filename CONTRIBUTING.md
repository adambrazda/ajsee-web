# Contributing to AJSEE

Děkujeme za váš zájem přispět k projektu AJSEE! 🙌

## 🧭 Jak začít

1. Forkněte tento repozitář
2. Vytvořte větev (`git checkout -b feature/nazev-funkce`)
3. Proveďte změny a commitujte (`git commit -m 'feat: popis funkce'`)
4. Pushněte do své větve (`git push origin feature/nazev-funkce`)
5. Vytvořte **pull request**

## 💡 Co přijímáme

- Opravy chyb
- Nové funkce
- Zlepšení přístupnosti
- Vylepšení výkonu nebo SEO
- Překlady do jiných jazyků

## 🎯 Styl kódu

- **HTML/JS/SCSS** – používej formátování s linterem (`npm run lint`)
- Pojmenování: `kebab-case` pro třídy, `camelCase` pro proměnné

## ✅ Před odesláním

- `npm run lint`
- `npm run build` ověř, že vše funguje
- Pull request musí být čitelný a stručně popsaný

Děkujeme! 🙏
